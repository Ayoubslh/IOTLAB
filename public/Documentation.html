Arduino Basics

1. What is Arduino?

Arduino is an open-source electronics platform based on easy-to-use hardware and software. It's widely used for building digital devices and interactive objects that can sense and control the physical world.

2. Key Components:

  • Microcontroller
  • Digital Pins (0–13)
  • Analog Pins (A0–A5)
  • USB Port
  • Power Jack
  • Reset Button

3. Basic Tools Needed

  • Arduino board (Uno, Mega, etc.)
  • USB cable
  • Jumper wires
  • LEDs, resistors, sensors (e.g., temperature or motion)

Don’t worry — we provide all of these on our platform, so you can get started right away without needing physical components

Example: Blink an LED

This simple code makes the built-in LED on pin 6 blink on and off every second.

// LED on pin 6 blink on and off every second.
void setup() {
  pinMode(6, OUTPUT); // Set pin 6 as output
}

void loop() {
  digitalWrite(6, HIGH); // LED on
  delay(1000);           // Wait 1 second
  digitalWrite(6, LOW);  // LED off
  delay(1000);           // Wait 1 second

}
